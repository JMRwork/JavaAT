AT - Desenvolvimento de Serviços Web e Testes com Java - Prof:Elberth Moraes.

O projeto aqui desenvolvido foi estruturado atraves do framework Spark Java. Este framework se destaca pela sua simplicidade e produtividade, permitindo que os desenvolvedores construam APIs de forma rápida e eficiente, sem a necessidade de configurações complexas. Além disso, sua natureza leve e ágil o torna ideal para projetos que exigem desempenho e escalabilidade, enquanto sua flexibilidade e modularidade oferecem liberdade para adaptar a estrutura conforme necessário.
